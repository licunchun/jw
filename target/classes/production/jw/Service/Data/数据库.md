User学生老师管理员共有属性

```
"ID",
"name",
"password",
```

对应格式分别由`IDUtil`,`NameUtil`,`PasswordUtil`来确定格式



Student数据格式

```
"ID",
"name",
"password",
//枚举转字符串
"grade",
"gender",
"school",
//double转字符串
"money",
//CodeUtil和TimeUtil
"classes",
"times"
```

所有属性根据主键`ID`提供set和get方法
